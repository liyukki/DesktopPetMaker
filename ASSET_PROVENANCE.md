# Asset Provenance

## Public Application Brand

The public beta `resources/branding/` family is original geometric artwork
created for Desktop Pet Maker. It combines a yellow rounded application tile,
a white creation spark, a charcoal desktop-pet face, and cyan digital eyes.
It does not depict or derive from a third-party character.

The SVG source and deterministic Pillow generator are included in the
repository. Redistribution and modification with Desktop Pet Maker are
permitted.

## Character And Pet Assets

The public source and runtime packages contain no user character projects.
Users are responsible for confirming the rights to artwork, models, fonts,
voices, prompts, and other media they import or distribute.

## Optional Original Sample

`DesktopPet-Original-Robot-Sample.petpack` is an optional geometric robot
sample created for this project. It is distributed separately and may be
modified and redistributed with Desktop Pet Maker.
